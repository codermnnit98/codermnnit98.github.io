Guess the number?

Rules

  1. The number can be between 1-100
  2. You have 7 tries to guess the number
  3. Each time you guess the number, the emulator answers with 2 choices

    4. 1: If the choosen number is less than the number to be guessed

    2: If the choosen number is greater than the number to be guessed

  4. A window will appear displaying "You have lost the game" if the number of tries exceeds 7
  5. A window will appear displaying "You have won the game" if you have gussed the number correctly
